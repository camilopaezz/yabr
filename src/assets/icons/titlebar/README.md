# Titlebar icons

SVGs from **Lucide** ([lucide-static](https://github.com/lucide-icons/lucide) v0.469.0), ISC license.

Vendored as static assets (not an npm dependency). Used only by the custom window titlebar.

| File | Lucide name | Role |
|------|-------------|------|
| `settings.svg` | `settings` | Open settings |
| `minus.svg` | `minus` | Minimize |
| `square.svg` | `square` | Maximize |
| `copy.svg` | `copy` | Restore (overlapping frames) |
| `x.svg` | `x` | Close |
